针对JavaWeb 中css 的一些了解

首先css是指层叠样式表 (Cascading Style Sheets)
就是用来修饰html中的元素的一种方法。
用css可以极大的提高效率
一般有两种形式。
一种是写在html页面的head里 叫做内部样式表
另一种专门写在一个css文件中 叫做外部样式表
还有一种 可以写在html元素内部的叫做内联样式

外部样式表的引用格式是

对于css来说 作用于哪些元素上便于区分，也有一个选择器

首先可以对于一个元素的所有引用都使用 叫做元素选择器。

第二种叫做id选择器
使用方法为$ # + id名$
就可以针对同一id名的元素进行样式定义（但是要注意的一个元素的id应该是唯一的。)

第三种叫做类选择器
使用方法为$. + 类名$
这里就可以对多个元素进行修改

当然还有更准确的选择就是 元素名+类选择器一起来选择

一般比较常见的css 样式属性

width 宽度
background-color 背景颜色
background-image:url 图片作为背景
background-size: contain 背景平铺

文本
color 颜色
text-align 对齐方式
text-decoration 文本修饰 overline line-through underline 上划线 删除线 下划线
还有一个是 针对超链的 none 是去掉下划线

line-height 行间距

对于字体

font-size 大小
font-size 风格 italic 斜体
font-weight 粗细 bold 粗体
font-family 字库
当然可以声明到一起

表格边框想只有一个可以设置一个 border-collapse

对于边框来说
border-style 风格 solid 实现 dotted 点状 dashed 虚线 double 双线
border-color 颜色
border-weight 宽度
当然跟字体一样可以声明到一起

padding-left 左内边距 同理上下左右

margin-left 左外边距 同理上下左右

隐藏元素也有两种方式
display:none
visibility:hidden
