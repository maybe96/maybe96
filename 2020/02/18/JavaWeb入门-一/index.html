JavaWeb入门(一)

对于html的学习。
首先是基本的html格式

然后对于一些基本元素的总结
标题 从h1——h6 由大到下
段落 p 在段落中的文本会自动换行
粗体 strong b 推荐使用strong
斜体 i em
还有一个 font 但是不推荐使用，使用样式对字体的颜色大小等进行修改是主流
删除线 del
下划线 ins

然后是一个简单的使用

接下来是进阶一点元素介绍

对于图像的调用 img src = “文件目录” alt = “当图片不可用时，显示的话”
文件目录分几种情况
1.同级直接使用文件名即可
2.上级需要加../
3.使用绝对路径时，使用file://

超链接
a href=”跳转到的页面地址” target=”_blank” (表示在新的页面显示超链) title = “显示提示文字”
a 表示显示超链

表格 table
tr表示一行 td表示一行中的一列
格式一般如下

列表
ui 无序列表 ol 有序列表
li 表示列表中的元素

iframe 表示 内联框架 内嵌一个页面

进阶练习

实际成品

在进阶一点，进阶到表单
表单里比较重要的就是form
form就是创建一个html表单，在表单里面可以创建文本框，复选框，提交按钮等，主要就是用于向服务器传输数据
属性一般有
action 是将数据传送到jsp中，提交到jsp页面的地址
method 传送数据的形式 1.get 在地址栏能看到传送的参数
2.post 看不到上传的参数，可以上传文件

然后表单里的一些元素根据input中的属性type进行区分
比较常用的
text 文本框 placeholder 为 背景文字 value 为 默认值
password 密码框 输入显示为星号
radio 单选框 check = “checked”为默认选中 name 为命名分组
checkbox 复选框 与单选框属性类似
select 下拉列表 option 是列表中的元素 multiple=”multiple” 为多选 selected=”selected” 默认选中。
textarea 文本域 row cols 长宽
button 按钮 不具备form 的功能
submit 提交按钮
reset 重置按钮 将同一个表单中的文本清空

一个进阶的练习
