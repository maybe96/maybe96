基础的web服务器设置

安装必要的web服务

yum -y install httpd
yum -y install php

配置域名

首先了解web服务器保存的位置 /etc/htppd
然后是配置管理类 /etc/httpd/conf/http.conf文件 是http服务器的主要配置文件
内容方面
默认的位置是 /var/www
/var/www/html/ 根目录
/var/www/error/ 错误信息
/var/www/icons/ 网页图标
/var/www/usage/ 服务器运行情况
对于错误信息和访问信息的访问日志存放在 /var/log/httpd

直接配置Apache

在根目录(/var/www/html/) 中 添加 index.php.
然后再里面添加一些前端代码，类似

然后直接启动Apache，然后关闭防火墙，就可以在内网和外网中看到测试结果
systemctl restart httpd
systemctl stop firewalld
setenforce 0

修改根目录

如果不希望使用Apache根目录中的web页面，可以修改根目录。
即创建了一个www的文件夹，然后加入index.html
然后在/etc/httpd/conf/httpd.conf中修改配置文件
即在DocmentRoot , Directory ,以及下方的Directory 的三个根目录的地址改为自己设置的地址就可以。

修改个人目录

首先创建 用户
useradd 用户名 (crs)
然后创建 用户web文件夹 (/home/crs/www)
这里最好不要创建到root根目录，需要修改权限
chmod 755 /home/crs
chmod 755 /home/crs/www
chown crs:crs /home/crs/www

然后在Apache中修改配置文件(/etc/httpd/conf/httpd.conf)

然后关闭防火墙，重启httpd服务即可
访问地址为 IP地址/crs （192.168.36.100/crs）
